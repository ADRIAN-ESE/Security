## 🎯 Overview

**Orion Security** is an ambitious, multi-module cybersecurity platform designed for:

✅ **Security Students & Learners** - Master cryptography, network security, and offensive/defensive techniques  
✅ **Penetration Testers** - Leverage pre-built tools for reconnaissance, scanning, and exploitation  
✅ **Security Engineers** - Understand vulnerability detection and system hardening  
✅ **Ethical Hackers** - Develop practical skills through hands-on projects  

The suite includes **30+ projects** spanning:
- Cryptography & Encryption
- Password Security & Cracking
- Network Scanning & Reconnaissance
- Vulnerability Assessment
- Authentication Systems
- File Integrity Monitoring
- Digital Forensics
- System Hardening

## 📦 Installation & Setup

### **Prerequisites**

```bash
# System Requirements
- Python 3.8+
- Node.js 14+ (for JavaScript modules)
- pip package manager
- Virtual environment (recommended)
```

### **Clone the Repository**

```bash
git clone https://github.com/ADRIAN-ESE/Security.git
cd Security
```

### **Python Environment Setup**

```bash
# Create virtual environment
python -m venv venv

# Activate virtual environment
# On Windows:
venv\Scripts\activate
# On Linux/macOS:
source venv/bin/activate

# Install required packages
pip install -r requirements.txt  # If available
# Or install individually as needed
pip install cryptography
pip install scapy
pip install requests
pip install opencv-python
pip install pynput
```

### **JavaScript Setup**

```bash
# Navigate to JavaScript module directories
cd "Cryptography JS"
# Open index.html in a modern web browser
# Or use a local server:
python -m http.server 8000
```

## 🎯 Quick Start

### **1. Basic Encryption**
```bash
cd Cryptography/Message
python main.py
# Follow prompts to encrypt/decrypt files
```

### **2. Password Strength Test**
```bash
cd Password_Tester
python "Password Strength Test.py"
# Input passwords to evaluate security
```

### **3. Port Scanner**
```bash
cd "Custom Vulnerability Scanner"
python port_scanner.py <target_host>
# Scan target for open ports
```

### **4. Password Cracker (Dictionary Attack)**
```bash
cd CRACKER
python password_cracker.py --wordlist wordlist.txt --target <hash>
```

### **5. Forensic Analysis**
```bash
cd Forensics
python "Browser Analyze.py"
# Extract and analyze browser history
```

### **6. Web-Based Tools**
```bash
cd "Cryptography JS"
# Open index.html in browser
# Use interactive encryption/decryption interface
```

## 🐛 Troubleshooting

### **Common Issues**

**Issue:** Module not found error
```bash
Solution: pip install -r requirements.txt
```

**Issue:** Permission denied on Linux
```bash
Solution: chmod +x script_name.py
```

**Issue:** Port already in use
```bash
Solution: Change port number or kill process using the port
```

**Issue:** OpenCV installation fails
```bash
Solution: pip install opencv-python-headless
```
