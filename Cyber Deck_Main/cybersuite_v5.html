 Algorithm Reference
🛡️ AES-256
Industry-standard symmetric encryption. Requires a passphrase. Output is base64-encoded ciphertext. Best choice for any sensitive data.
⚡ XOR Cipher
Bitwise XOR of each character with an integer key (0–255). Output is base64-encoded. Fast but security depends entirely on key secrecy.
🏛️ Caesar Cipher
Classical shift cipher — moves each letter N positions in the alphabet. Educational use only; trivially broken by modern methods.
📦 Base64
Encoding, not encryption. Converts any text/binary into ASCII-safe strings. No key needed — use for data transport, never for secrecy.
