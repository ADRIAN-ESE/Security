 Algorithm Reference
AES-256
Industry-standard symmetric encryption. Requires a passphrase. Secure for sensitive data in transit or at rest.
XOR Cipher
Bitwise XOR with an integer key (0–255). Fast and simple — output base64-encoded for safe transport.
Caesar Cipher
Classical letter-shift cipher. Shift value 1–25. Educational use only — not secure for sensitive data.
Base64
Encoding (not encryption) — converts binary data to ASCII text. No key needed. Reversible by anyone.
